# Doctor Appointment Booking System

A full-stack Doctor Appointment Booking System with **role-based access control**, enabling patients, doctors, and administrators to manage appointments efficiently. The platform supports real-time scheduling, secure authentication, and streamlined clinic operations.

---

## Features

- 🔐**Authentication & Authorization**
  - Secure login/signup system
  - Role-based access (Patient, Doctor, Admin)

- 👨‍⚕️**Patient Module**
  - Browse doctors
  - Book, reschedule, and cancel appointments
  - View appointment history

- 🩺**Doctor Module**
  - Manage availability & time slots
  - View scheduled appointments

- 🛠️**Admin Module**
  - Manage doctors and users
  - Monitor system activity

- ⏱️**Real-Time Scheduling**
  - Dynamic slot availability
  - Prevents double booking

---

## 🧑‍💻Tech Stack

- **Frontend:** React.js / HTML / CSS / JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB / MySQL  
- **Authentication:** JWT

## ⚙️ Installation & Setup

# Clone the repository
git clone https://github.com/minnoli-madhyan/Doctor-Appointment-Booking-System.git

# Navigate to project folder
cd Doctor-Appointment-Booking-System

---

### 🔧 Backend Setup

cd backend  
npm install  
npm run server  

---

### 🎨 Frontend Setup

cd frontend  
npm install  
npm run dev  

---

### 🛠️ Admin Panel Setup

cd admin  
npm install  
npm run dev
